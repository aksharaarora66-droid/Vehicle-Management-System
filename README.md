# Vehicle-Management-System
The project uses Hierarchical Inheritance, where multiple classes branch out from the same base.

Vehicle (Base Class): Common attributes for all transport (ID, Brand, Model, etc.).

MotorVehicle (Subclass of Vehicle): Specific to vehicles with engines (Cars, Bicycle).

Bicycle (Subclass of Vehicle): Specific to human-powered vehicles. It adds attribute bike_type (e.g., Mountain, Road).
